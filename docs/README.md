The Silver stack is a self-hosted product baseline with a Go API server, embedded dashboard, JSON and OTLP HTTP JSON/protobuf telemetry ingestion, login sessions, scoped tokens, PostgreSQL-backed runtime persistence in the Podman stack, ClickHouse telemetry archival and query APIs, alert rules, notifications, local uptime checks, Helm deployment assets, and a live demo checkout application.
